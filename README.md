# 🐕 KinologApp

Информационная система для кинологического центра, написанная на Java.

## ✨ Основные функции
* **Учет клиентов и собак**: Хранение данных о владельцах и их питомцах.
* **Финансовый модуль**: Расчет стоимости услуг с учетом VIP-статуса (точность через `BigDecimal`).
- **Хранение данных**: Реализован паттерн `Repository` для хранения состояния объектов в памяти (In-memory DB).
- **Контентная база**: Система управления статьями и видеоуроками.

## 🛠 Технологии
- **Java 23** (использование Pattern Matching для `instanceof`).
- **Слоистая архитектура**: Четкое разделение на `model`, `repository`, `enums`.
- **Git**: Версионность и чистая история коммитов.

## 🚀 Как запустить
1. Склонируйте репозиторий.
2. Откройте проект в **IntelliJ IDEA**.
3. Дождитесь загрузки зависимостей.
4. Запустите класс `com.kinologapp.Main`.
